# STO Rules

Краткий справочник правил оформления по `СК-СТО-ТР-04-1.005-2015`

> [Скачать PDF](https://raw.githubusercontent.com/thebandik/sto-rules/main/sto-rules.pdf)

## Web справочник

> [Открыть web справочник](https://thebandik.github.io/sto-rules)

В web справочнике можно открыть конкретное правило или пример через ссылку с кодом в `#hash`

Примеры: [Правило P09-06](https://thebandik.github.io/sto-rules/#P09-06), [Пример E02-01](https://thebandik.github.io/sto-rules/#E02-01)

Можно передать сразу несколько кодов через запятую: [E02-01, P03-04, P03-15](https://thebandik.github.io/sto-rules/#E02-01,P03-04,P03-15)

Поддерживаются коды:

- `P..-..` - правила и параметры
- `E..-..` - примеры

Также работают query-ссылки вида `?rule=P09-06` и `?rules=E02-01,P03-04`
